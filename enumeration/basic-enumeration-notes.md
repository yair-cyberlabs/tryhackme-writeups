# Enumeration Basics

## Objective
Understand how to discover information about a target system.

## Enumeration
Commands used to gather information.

nmap -sC -sV target-ip

## Exploitation
Attempt to find vulnerabilities based on enumeration results.

## Privilege Escalation
Check for misconfigurations or credentials that allow elevated access.

## Capture the Flag
Document the flag retrieved during the exercise.

## Lessons Learned
- Enumeration reveals system services
- Nmap helps identify open ports
- Information gathering is critical before exploitation
